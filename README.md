# foodie_blog
맛집 블로그

## 1. 목표 및 기능

### 1.1 목표
- django에 익숙해지기
- 맛집 정보와 직접 만든 음식을 공유하는 커뮤니티 사이트

### 1.2 기능
1. 메인페이지 구현 (완료)
2. 게시글 작성 (완료)
3. 게시글 상세보기 (완료)
4. 게시글 수정 (완료)
5. 게시글 삭제 (완료)
6. 게시글 검색
7. 회원가입
8. 로그인
9. 사진 업로드

## 2. 개발 환경
- Python - 3.9.10
- django - 4.2.3
- sqlite3 - 3.35.5
